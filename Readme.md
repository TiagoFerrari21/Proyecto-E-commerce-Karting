# E-Commerce & Reservas - Kartódromo 🏎️

Trabajo Práctico N° 1 – Git y Docker para la materia **Aplicaciones Web Cliente e Ingenieria de Software** (ISTEA).

## 📝 Descripción del Proyecto
Este proyecto consiste en una plataforma web estática diseñada para un **Kartódromo**. El sitio web combina dos funcionalidades principales para los amantes del karting:
1. **E-Commerce:** Un catálogo interactivo de artículos, repuestos y accesorios de karting con un sistema de carrito de compras gestionado en el lado del cliente.
2. **Sistema de Turnos:** Un formulario integrado donde los usuarios pueden solicitar y agendar un turno para alquilar karts y usar la pista del kartódromo.

Toda la aplicación ha sido empaquetada y dockerizada utilizando un servidor web **Nginx** ligero para garantizar que corra exactamente igual en cualquier computadora.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** HTML5, CSS3 y JavaScript (Vanilla JS).
* **Servidor Web:** Nginx (Imagen oficial basada en Alpine Linux).
* **Contenedores:** Docker.
* **Control de Versiones:** Git y GitHub.

## 📌 Requisitos Previos
Antes de ejecutar el proyecto, asegúrate de tener instalado en tu sistema:
1. **Git** (para la clonación del repositorio).
2. **Docker Desktop** (con el servicio activo).
